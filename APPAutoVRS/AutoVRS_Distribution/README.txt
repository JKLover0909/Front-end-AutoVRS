    AutoVRS - Automatic Visual Recognition System
=====================================================

HƯỚNG DẪN SỬ DỤNG
================

1. CÁCH CHẠY ỨNG DỤNG:
   - Nhấp đúp vào file "autovrs_app.exe" để khởi động ứng dụng
   - Đảm bảo các file sau luôn ở cùng thư mục:
     * autovrs_app.exe (file chính)
     * flutter_windows.dll
     * data/ (thư mục chứa tài nguyên)

2. ĐĂNG NHẬP:
   - Mật khẩu Worker: "worker"
   - Mật khẩu Admin: "admin"
   - Admin có quyền truy cập tất cả tính năng
   - Worker chỉ truy cập được các tính năng VRS

3. CÁC TÍNH NĂNG CHÍNH:
   ✓ Quản lý Model sản phẩm
   ✓ Giám sát Auto VRS
   ✓ VRS Thủ công với các công cụ:
     - Định vị vị trí Bo (4 bước)
     - Điều chỉnh ánh sáng
   ✓ Thống kê chi tiết:
     - Tỉ lệ phán định NG/OK
     - Phân tích loại lỗi với biểu đồ

4. YÊU CẦU HỆ THỐNG:
   - Windows 10/11 (64-bit)
   - RAM: Tối thiểu 4GB
   - Dung lượng: ~100MB
